# Priyanka Learning Hub 📚

Priyanka Learning Hub is an online notes management and selling platform developed using Java Full Stack technologies. Users can register, login, view notes, and access premium notes after payment.

## 🚀 Features

- User Registration and Login
- User Authentication
- Notes Management System
- Free and Premium Notes
- Premium Notes Unlock After Payment
- Admin Panel for Managing Notes
- Responsive User Interface

## 🛠️ Technologies Used

### Backend
- Java

- Maven
